## What is BEIDNA?
BEIDNA specifies a **triple-helix genome** for sovereign records by braiding three strands‚Äî**Identity, Behavior, Economy**‚Äîinto a single, machine-verifiable event record. Each event emits **Tri-Header Receipts** (IH/BH/EH), is anchored with CID, governed by **Admission Gate** (ethics & compliance), and protected by **FR2 bounded rollback** (deterministic corrections with prefunded reserves, no history deletion). Public invariants (header integrity, lineage continuity, reserve sufficiency, jurisdictional compliance) enable verification without private data exposure.

**Distinction vs. prior BEI filings:** earlier work focuses on workflows/clearing; **BEIDNA** defines the **encoding algebra** `encode | splice | fork | merge | roll | verify`, suitable for SEP/FRAND standardization.
